# Hotel Booking Cancellation Analysis

 Project Overview

This project analyzes hotel booking data to understand cancellation patterns and customer behavior.

 Dataset

* Source: hotel_bookings.csv
* Contains booking details like customer type, hotel type, cancellations, etc.

 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Dashboard

 Key Analysis

* Cancellation rate by hotel type
* Customer behavior insights
* Booking trends

 Insights

* City hotels have higher cancellation rates
* Certain customer segments cancel more frequently
* Seasonal trends observed in bookings

 Conclusion

This analysis helps hotels improve booking strategies and reduce cancellations.
